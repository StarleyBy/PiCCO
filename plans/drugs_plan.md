# План создания файла drugs.html

## Основные изменения по сравнению с piccoworld.html:

1. Замена параметров ПИККО на лекарственные препараты
2. Создание уникального стиля для каждого препарата
3. Изменение структуры ячеек:
   - Верхняя часть: фармакологическая группа, подгруппа, название в Израиле
   - Центр: символ препарата и название крупным шрифтом
   - Нижняя часть: дозировки для взрослых/детей, пути введения
4. Изменение содержимого тултипов:
   - Механизм действия
   - Эффекты
   - Особенности
   - Противопоказания
   - Длительность действия
   - Интересные факты
   - Преимущества
   - Опасности
