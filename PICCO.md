# Проект "Онлайн-методичка PiCCO"

## Цель проекта
Создание "онлайн-методички" для медицинского персонала по работе с прибором PiCCO. Ресурс должен быть удобным для быстрого изучения, вспоминания или подсмотра необходимой информации.

## Принципы
*   Без авторизации и ввода данных.
*   Статичный веб-сайт (HTML, CSS, JavaScript).
*   Простая и понятная структура.
*   Оптимизация для максимальной читаемости и удобства использования.
*   Приоритет профессионального и чистого дизайна.
*   **Mobile-first:** Изначальная ориентация на экраны смартфонов.
