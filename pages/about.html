About the PiCCO Online Manual Project

This project is a amateur non-commercial, voluntary resource created for quick and convenient access to essential information about PiCCO (Pulse Contour Cardiac Output) monitoring and advanced hemodynamics.

Project Goal

The main goal of this resource is to condense complex physiology, clinical algorithms, and technical protocols of the PiCCO method into an easily digestible format (e.g., checklists, quick references, and tables). We aim to provide medical professionals - including intensivists, anesthesiologists, and ICU nurses - with rapid access to the core diagnostic and therapeutic principles of the PiCCO method at the point of care.

Important note: This manual serves as an educational and reference tool only. It is not intended to replace formal training, official medical guidelines, or professional clinical judgment. Always refer to local protocols and manufacturer documentation for definitive instructions on use and clinical decision-making.

Project Structure

The site is organized as a set of HTML pages, each dedicated to a specific aspect of PiCCO monitoring:

  • Introduction: General overview of the PiCCO method
  • Principles: Physics and physiology of the method
  • Equipment: Catheters, interface, and setup
  • Calibration: How calibration is performed
  • Indications and Contraindications: When to use and when not to use
  • Parameters: Detailed explanation of all parameters (GEDVI, EVLWI, SVV, etc.)
  • Interpretation: Algorithms and clinical scenarios
  • Cases: Real clinical examples
  • Errors: Common errors and pitfalls
  • Reference: Glossary and normal values
  • Frequently Asked Questions: Answers to common questions

Technical Features

The site is designed as a static website using HTML, CSS, and JavaScript to ensure maximum compatibility and performance:

  • Mobile-friendly: Site optimized for viewing on mobile devices (mobile-first)
  • Dark/light theme: Support for theme switching for comfortable reading
  • Search: Built-in search function across all sections
  • Print version: Special CSS styles for printing
  • Interactive elements: Checklists, diagrams, and calculators for enhanced engagement
